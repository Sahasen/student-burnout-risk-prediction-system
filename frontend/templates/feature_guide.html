⚠️ Data Leakage Fix: Wellbeing_Index Removed

The original dataset had a Wellbeing_Index column with correlation –0.81 with the Burnout_Risk target — this is classic data leakage (the target is partially encoded in the feature). It was removed from all models in this system. The model now uses 19 features instead of 20, making the predictions genuinely predictive rather than artificially inflated.
