📝 Data Cleaning Notes (Real Google Form Data)

Issues found in real responses:

  • Range strings like "5-6", "2-3 hours", "5to6", "5-6 hours" → parsed to midpoint (e.g., 5.5)
  • Noise values like "No", "0-2", "0-3" in gaming column → parsed as 0 or midpoint
  • Outliers: sleep=16 hours (impossible) → clipped to max 12. Gaming=30 hours → clipped to 12.
  • Trailing whitespace in country field ("India ") → stripped
  • Mixed types: numeric columns stored as object due to string entries → all re-parsed as float
  • Blank "motivation this week" column → dropped (all NaN)
